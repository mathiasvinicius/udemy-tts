# udemy-tts

Extensão para Edge/Chrome que lê em voz alta as legendas do player da Udemy em tempo real com vozes nativas do navegador e ajuste automático de velocidade para reduzir atraso.

## Status atual

- Modo resumo/salvamento local: desativado temporariamente.
- Foco: leitura TTS da legenda do player + vozes nativas do navegador.
