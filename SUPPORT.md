# Support - Udemy Transcript TTS (Free)

## Quick checks

1. Open a Udemy lesson page (`https://*.udemy.com/*`).
2. Ensure subtitles are enabled in the player.
3. Open the extension popup and set:
   - `Ativar` = on
   - `Fonte` = `Legenda do player` (recommended default)
4. Pick a voice and test volume/rate.

## Common issues

- "Abra uma aula da Udemy para usar."
  - Open an active Udemy lesson tab and retry.

- No voice list
  - Reload the lesson tab and reopen the popup.

- Delay in reading
  - Enable automatic mode and tune min/max speed in settings.

## Report a bug

Create an issue with:
- Browser version (Edge/Chrome)
- Extension version
- Reproduction steps
- Debug log excerpt (if enabled)
